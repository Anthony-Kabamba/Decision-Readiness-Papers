# Papers

This directory contains the published papers in the **Decision Readiness** series.

Each paper examines a specific source of decision failure observed in industrial or organisational environments and applies a consistent analytical framework to understand how system design influences decision quality.

The papers are intended to be read independently while contributing to a broader body of work on Decision Readiness.

## Structure

Each paper follows the same structure:

1. Context
2. Observed Structural Pattern
3. Decision Reframe
4. Decision Failure Mode
5. Accountability Consequence
6. Decision Readiness Assessment
7. Diagnostic Question

This structure promotes consistency, comparability, and repeatability across analyses.

## Reading Order

Although each paper is self-contained, the recommended reading order is:

1. Information Visibility
2. Decision Capacity
3. Signal Integrity
4. Information Capture

Together, these papers introduce the foundational concepts of Decision Readiness while examining different sources of decision degradation within complex operational systems.

## Citation

If referencing a paper, please cite the individual document rather than this directory.

## Note

These papers are analytical works intended to stimulate discussion around system design, information quality, and decision support.

They are independent publications and should not be interpreted as evaluations of any specific organisation, employer, operational site, or individual.
